**GRADE CALCULATION**

This project implements a Student Grade Calculator using Python.
It calculates the average marks of 5 subjects and assigns a grade based on predefined criteria.

The project also includes unit testing using pytest and a CI pipeline using GitHub Actions to automatically run tests whenever code is pushed to the repository.

**Grade Criteria**:
Average Marks	Grade
- ≥ 90	A+
- ≥ 75	A
- ≥ 60	B
- ≥ 50	C
- < 50	Fail

**Technologies Used**: 
- Python 
- Pytest (Unit Testing)
- GitHub Actions (CI/CD)
